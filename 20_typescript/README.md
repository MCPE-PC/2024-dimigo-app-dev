# TypeScript

- 자바스크립트에 타입을 부여한 프로그래밍 언어

- 자바스크립트의 슈퍼셋

- MS에서 개발, 유지보수하며 엄격한 문법을 지원

- 클라이언트 사이드 + 서버 사이드 모두 개발

- 타입스크립트 -> 컴파일 -> 자바스크립트 파일

## 장점

- 컴파일 타임에 타입 체크 -> 에러 사전 방지

- 동적 타이핑보다 실행 속도가 빠름

- 강력한 생태계를 가짐, 많은 라이브러리들이 지원

- 개발 도구의 강력한 지원 -> 개발 생산성 향상

## TypeScript 설치

- `npm init`

- `npm i -D typescript ts-node`

- `npx tsc -v`
